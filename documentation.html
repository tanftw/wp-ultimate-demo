1. Installation - top

Site Requirements

WP Ultimate Demo has tested and compatibility with

  • WordPress 3.9+
  • PHP 5.2+
  • It's recommended that you not run your demo website on same server with your other production websites.

Preinstall

  • Make sure you have backed up your database before install

Installation

  1. Download wp-ultimate-demo.zip and unzip the package
  2. Upload wp-ultimate-demo folder to the wp-content/plugins/ directory
  3. Activate the plugin through the Plugins menu in WordPress
  4. Start using the plugin through the menu Settings\WP Ultimate Demo

2. Configuration - top

To change WP Ultimate Demo settings, simply go to Dashboard\Settings\WP Ultimate Demo

General Settings

Offline Mode

After plugin setup, Offline Mode is turned on to wait you change settings on the first run. When in offline mode, users with specified roles can modify and update your site (of course files, plugins and themes also if they have permission). Otherwise, when Offline Mode is turned off, the cleanup will clean all changes in your specified time (default: each 24 hours).

If you want to change this plugin settings, you have to switch to Offline Mode.

In short, uncheck Offline Mode when you want to publish the demo and check Offline Mode when you want to make changes.

Security

There're 3 options in this section which turned on by default while they are recommended settings.

Don't let users modify your files, upgrade website, themes, and plugins
This will stop your users from change your files, update website, themes and plugins like that description whilst Offline Mode is turning on.
If you already use CHMOD, set wp-config.php constants, or use other plugins and methods to prevent users modify your files, you can ignore this option.

Hide this plugin from any one except me
If this option is turned off, anyone with Administrator role can access this area, switch WP Ultimate Demo to Offline Mode and make changes. So it's recommended that you always turn on this option or check permission carefully when you give your demo users credentials.

Hide admin menu
Since 1.1, you can list all menu items which you want to hide to this text box separated by commas. It will hide menu items from all users except you, and if they try to access those menu via URL, it will redirect to /wp-admin

Data Cleanup

By default, WP Ultimate Demo cleanups your database each day at 00:00. But you can change this option to make cleanup runs more or less frequently. For example, if you set to 2, that means this will run each two hours.

Countdown

Countdown is a cool feature of WP Ultimate Demo which displays a countdown bar with your custom message template if users session is going to expired in x seconds. Please remember place %time% variable so countdown can works properly.

Login Settings

Auto Login
This option contains Disable, Enable and Prefill.
  • Disable: Nothing happened, you give your users credentials, then they use it to login to your website.
  • Enable: You can select an user, then anyone can bypass login form with this credential. Remember that you can still use login form to login with other user.
  • Prefill: You fill the user name and password, then the login form will auto fill this credential by default.
Login Message

Like this title and description, you can put your custom message in the top of login form below your logo to make more impressive for your users.

Manual Cleanup Data

Hit Cleanup button, all changes in your demo website are reverted back to your time you edit in Offline Mode.

3. Advanced Configurations - top

Toggle Offline Mode Advanced

Sometimes, WP Ultimate Demo can't touch to your files to toggle offline mode because your files permissions. You can still change this setting by creating .demo or .demo_disabled in your wp root directory represent the plugin state. .demo means Offline mode is turn on .demo_disabled mean Offline mode is turned off

Detect Offline Mode

If you want to check Offline Mode in your theme or plugin, just use wud_is_demo_active() function. This will return true when Offline mode is turned on, otherwise return false

4. Uninstallation - top

Before uninstall, make sure you've backed up your website.

If you've set Hide this plugin from any one except me option in Security section. Make sure you're accessed with right user to deactivate this plugin.

Go to Plugins\Installed Plugins, locate to WP Ultimate Demo and click "Deactivate". This will cleanup all demo data, remove all WP Ultimate Demo settings and turn off demo.
